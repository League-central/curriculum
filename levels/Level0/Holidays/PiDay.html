Pi Day

1. Create a class called PiDay and give it a main method.

2. Make a String variable called pi and initialize it with some of the digits of Pi.
     If you don't know Pi, go to http://www.piday.org/million/ to get as many digits as you like.
     Just make sure you have at least 10!

3. Print out 4 of the digits of your string. Use a for loop!
     HINT: The first value is pi.charAt(0), the second is pi.charAt(1)......
     Run your code and you should see 3.14 in the Console.

4. In a popup, ask the user for the next digit of Pi. Check to see if it is correct. HINT: use Character.toString()

     If they get it right, print the digit to the console, congratulate them, then ask for the next
     If they get it wrong, tell them "NO PIE FOR YOU!" and end the program HINT: System.exit(0);

5. Keep asking them for each new digit until they make a mistake or you run out of digits in your String
     HINT: use pi.length()

6. If they correctly guess all 10 digits, the Console should show 3.1415926536 and you can show them fun facts about pi (https://www.youtube.com/watch?v=6TPjRoWm8Ck) using the method below.

static void playVideo(String videoURL) {
     try {
          URI uri = new URI(videoURL);
          java.awt.Desktop.getDesktop().browse(uri);
     } catch (Exception e) {
          e.printStackTrace();
     }
}
