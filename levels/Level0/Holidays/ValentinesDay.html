Valentine's Day Fun

Here are a couple of choices for a Valentine's Day recipe.
Whichever recipe you choose, begin by finding a background image with a Valentine's design like this one:

1. A Robot Valentine

Code a Robot to draw a heart. Put the Valentine's Day image in the background.
It will look something like this:


If you have extra time, write a message on the code with the robot. One idea is putting and "I" on the left and a "U" on the right of the heart.

2. A Nerdy Valentine

In Processing, start a new sketch. Make the Valentine's Day image the background
Convert each character in the String "Happy Valentine's Day" to its number code and print it on the background to make a nerdy Valentine's Day card.

Use your own code if you can, or use the helper method included below:

It will look something like this:


String convertToNumbers(String in) {
      String out = "";
      for (int i=0; i < in.length(); i++) {
             int num = (int) in.charAt(i);
             out += num;
              out += " ";
      }
      return out; }
