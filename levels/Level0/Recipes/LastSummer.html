I Know What You Did Last Summer

Goal:

Tell the user that you know what they did last summer!

Steps:

  1. Create a class called LastSummer.
  2. Add a main method.
  3. Use JOptionPane.showInputDialog to ask them their name.

    Remember: Use a variable to store their name
  4. Use JOptionPane.showMessageDialog to tell them you know what they did last summer.
    (e.g. "I know that you [activity] last summer, [name]. Muhahaha!")
