Random Checkpoint

On paper, write code that does the following:


1. Get a random whole number that is less than 1,000.










2. Get another random whole number that is less than 900.










3. Print:

          a) An addition problem using the above random values
               e.g. "99 + 64 = 163"
          b) A subtraction problem using the above random values
                e.g. "99 - 64 = 35"
