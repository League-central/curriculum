Tasty Tomato

Goal:

Draw a tomato and take a bite out of it!

Steps:

  1. Create a new Processing sketch and copy and paste the code below: 
    void setup() {
    size(500, 500);
}
void draw() {
    background(200, 200, 200);
    noStroke();
    ellipse(150, 200, 150, 150);
    ellipse(212, 200, 150, 150);
    rect(176, 103, 12, 32);
}
  2. Make the tomato red.
  3. Make the stem green.
  4. Take a bite out of the tomato.
  5. Only bite the tomato when the mouse is pressed
