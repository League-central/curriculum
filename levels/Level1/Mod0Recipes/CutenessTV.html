Cuteness TV


Goal: Make a program that uses Model-View-Controller (MVC) design architecture.

The program will have a window (frame) with a number of buttons that will show different videos when clicked.


    If you need help with any of these steps, take a look at the previous recipes in this module.

    All the classes for this program are in the cuteness_tv package.


    Complete the Model


  1. Use the code below if you want the cutest videos ever. 
    String ducks = "https://www.youtube.com/watch?v=MtN1YnoL46Q";
String frog ="https://www.youtube.com/watch?v=cBkWhkAZ9ds";
String unicorns = "https://www.youtube.com/watch?v=a-xWhG4UU_Y";

    Note: If any of these video links no longer work, go to youtube and get cute replacements. Use variable names that give you a hint about what the videos contain.

    2. Make the View


  2. Find the CutenessTV class and follow the instructions to create the View for this program. NOTE: you will need a panel because there will be more than one button.

    3. Runner


  3. Find the CutenessRunner class and follow the instructions to copmlete it. Run it to make sure it is showing your frame with buttons before proceeding.

    4. Connect a Controller


  4. Find the CutenessController class and follow the instructions to copmlete it.
  5. Run the CutenessRunner and check that each of the buttons plays the right video.
