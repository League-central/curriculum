# piaf-vue

## Project setup
```
cd <project-root-dir>
npm install
```

### Compiles and hot-reloads for development
```
npm run dev
```

### Compiles and minifies for production
```
npm run build
```

### Run test server for production build
```
npm install -g serve
serve -s dist
```
